# 陈义 | 技术博客

## 项目介绍
这是一个个人技术博客网站，包含简历、博客、项目和联系页面。网站采用现代化的设计风格，具有响应式布局，支持主题切换功能。

## 技术栈
- HTML5 + CSS3 + JavaScript
- Tailwind CSS
- 模块化JavaScript架构
- 响应式设计

## 功能特性
1. **主题切换**：支持明暗主题切换，使用localStorage持久化存储
2. **粒子背景**：动态粒子背景效果，根据设备性能自动调整
3. **响应式设计**：适配不同设备尺寸，从手机到桌面端
4. **平滑动画**：卡片悬停效果、按钮动画、计数器动画等
5. **博客分类**：支持按分类筛选博客文章
6. **技能展示**：可视化技能熟练度展示
7. **联系功能**：包含邮箱复制、微信二维码等

## 页面结构
- `index.html`：主页，包含个人信息和模块导航
- `resume.html`：简历页面，展示工作经验和技能
- `blog.html`：博客页面，展示技术文章
- `projects.html`：项目页面，展示个人项目
- `contact.html`：联系页面，包含联系方式


## 部署指南
1. 克隆项目到本地
2. 启动本地服务器：`python -m http.server 8080`
3. 访问 `http://localhost:8080` 查看网站

## 注意事项
- 网站使用了Tailwind CSS CDN，需要网络连接
- 粒子系统在移动设备上会自动调整性能
- 主题设置会保存在localStorage中
