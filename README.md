# 陈义的技术博客

这是一个个人技术博客网站项目，专注于 AGV 设备技术运维领域的知识分享和经验总结。

## 项目概述

**作者**: 陈义  
**职业**: AGV 设备技术运维工程师  
**项目类型**: 个人技术博客网站  
**技术栈**: HTML5, CSS3, JavaScript, TailwindCSS  

## 功能特性

### 🎨 视觉效果
- **动态粒子背景**: 使用 HTML5 Canvas 实现的动态粒子系统，支持主题切换
- **响应式设计**: 完美适配各种屏幕尺寸
- **主题切换**: 支持深色/浅色模式切换，记忆用户偏好
- **交互动画**: 鼠标跟随光效、悬停动画等丰富的交互体验

### 📝 内容模块
- **首页 (index.html)**: 个人介绍、技能展示、项目列表
- **博客 (blog.html)**: 技术文章列表和详情页
- **简历 (resume.html)**: 详细的工作经历和个人技能
- **联系 (contact.html)**: 联系方式和动态位置足迹可视化
- **项目 (projects.html)**: 个人项目展示
- **欢迎页 (welcome.html)**: 带打字机动画的欢迎界面

### 🌍 位置足迹可视化
- **动态城市展示**: 使用 Canvas 绘制个人访问过的城市轨迹
- **地理定位**: 按照实际地理位置展示城市分布
- **动态连接**: 城市间连线展示关联关系
- **主题适配**: 支持深色/浅色模式下的不同配色

### 📱 响应式适配
- **移动端优化**: 针对小屏幕设备进行专门优化
- **自适应布局**: 根据屏幕尺寸自动调整元素大小和间距
- **触摸友好**: 适合触屏设备的操作体验

## 技术架构

### 前端技术栈
- **HTML5**: 结构化内容标记
- **CSS3**: 使用 TailwindCSS 框架和自定义样式
- **JavaScript**: ES6+ 特性，模块化组织代码
- **Canvas API**: 实现动态粒子效果和城市足迹可视化

### 代码组织
```
src/
├── components/           # 可复用组件片段
│   ├── footer.html
│   ├── header.html
│   └── ...
├── scripts/             # JavaScript 模块
│   ├── modules/
│   │   ├── animations.js    # 动画效果
│   │   ├── cursor.js        # 鼠标光效
│   │   ├── particles.js     # 粒子系统
│   │   └── theme.js         # 主题管理
│   └── utils/
├── styles/              # 样式文件
│   ├── base/            # 基础样式
│   ├── components/      # 组件样式
│   └── utilities/       # 工具类样式
```

### 性能优化
- **资源预加载**: 关键资源预加载优化
